# Ruby interface to apt-pkg

## INSTALL

~~~ console
aptitude install libapt-pkg-dev ruby2.0-dev
gem install bundler
bundle install
~~~

## BUILD

~~~ console
bundle exec rake compile
~~~

## TEST

~~~ console
bundle exec rake test
~~~
